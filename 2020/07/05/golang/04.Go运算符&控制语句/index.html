Go运算符&控制语句

发表于 2020-07-05 | 分类于 Golang

1. 运算符

  • Go 中的运算符与 Java 类型，都是从左至右结合

注意：

  • Go 中保留了指针，但是在 Go 中不支持指针运算以及 “->” 运算符，而是直接采用 “.” 来操作指针对象的成员
  • Go 中 “&” 操作符表示取变量地址，“*” 操作符表示通过指针间接访问目标对象
  • Go 中默认值为 nil，而不是 null

2. 控制语句

2.1 if 语句
  • Go 中 if 语句的使用与 Java 类似，区别是不用小括号包裹 if 条件语句
  • Go 中 if 条件语句中初始化的变量，只能在 if 语句块中使用
2.2 for 语句
  • Go 中没有像 Java 的 while语句，for 语句功能包括 Java 的 for 和 while 功能
  • 初始化和步进表达式可以是多个值
  • 条件语句每次循环会被重新检查，不建议在条件语句中使用函数，尽量提前计算好条件并以变量和常量代替
  • Go 中 for 条件语句中初始化的变量，只能在 for 语句块中使用
2.3 switch 语句
  • 可以使用任何类型或表达式作为条件语句
  • 不需要break，条件符合执行完自动终止
  • 使用fallthrough语句，程序会强制执行下一个case而忽略其表达式是否正确
  • 如果需要一个匹配所有情况都不是的，可以使用default语句
  • 一个case支持多个匹配条件
2.4 goto， break， continue 语句
  • 三个关键字都可以配合标签使用
  • 配合标签名使用时需注意大小写区分
  • break，continue 配合标签可退出多层循环
以上所有示例可在 github 查看
