Go数组

发表于 2020-07-05 | 分类于 Golang

1. 概念

数组：同一类型的有序集合。数组元素可以是基本类型，也可以是引用类型等。一般使用切片代替数组。

2. 声明和初始化

3. 数组初始化的特殊用法

  • 数组声明或数组没有赋值的初始化时，数组会有默认值
  • 数组的初始化可以通过 var + =，也可以 :=
  • 数组初始化时可以指定某一位置的值
  • 数组初始化可以不指定 size，go 语言会自动根据数组元素个数判断数组大小

4. 数组比较

两数组 ==、!= 比较时，需要同样大小并且元素类型一致才可比较，否则将会报错

5. 指针数组、数组指针

  • 指针数组：元素为指针的数组
  • 数组指针：一个指向数组的指针，声明使用 * 符号，初始化使用 & 符号
  • 不管是普通数组、指针数组、数组指针，都可以通过 len() 函数计算该数组长度
  • 不管是普通数组、指针数组、数组指针，只要声明了，就可以通过 len() 函数计算该数组长度

6. 作为参数传递

  • 数组为值类型、指针数组为值类型、数组指针为引用类型
  • 数组指针获取和修改某一索引值时，可以不需要添加 * 符号

7. 使用 new 创建数组

不建议使用，不能初始化为确定值

8. 多维数组

  • 直接初始化多维数组时，最后一大括号换行添加 , 符号，不换行不添加
  • 使用 … 忽略数组大小时，非顶级数组大小不能忽略

以上所有示例可在 github 查看
