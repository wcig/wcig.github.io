Go数据类型

1. 数据类型

1.1 基础类型：
  • 数字（num）
  • 布尔型（boolean）
  • 字符串（string）
1.2 聚合类型：
  • 数组（array）
  • 结构体（struct）
1.3 引用类型
  • 指针（pointer）
  • 切片（slice）
  • map
  • 函数（func）
  • 通道（channel）
1.4 接口类型
  • interface

2. 基础类型

2.1 布尔类型

布尔类型有：true/false 2种

2.2 字符串类型

字符串是固定长度的字符序列，Go 语言的字符串的字节使用 UTF-8 编码标识 Unicode 文本。

2.3 数字类型

数字类型声明后默认初始化为0

浮点型转成整型时，会舍弃小数部分，趋零截断（正数向下取整，负数向上取整）

常用整型：

类型 描述
int 有符号 32/64 位整数（根据系统定）
uint 无符号 32/64 位整数（根据系统定）
int8 有符号 8 位整型（-128 ~ 127）
uint8 无符号 8 位整型（0 ~ 255）
int16 有符号 16 位整型（-216 ~ 216-1）
uint16 无符号 16 位整型（0 ~ 232-1）
int32 有符号 32 位整型
uint32 无符号 32 位整型
int64 有符号 64 位整型
uint64 无符号 64 位整型

浮点型：

类型 描述
float32 IEEE-754 32位浮点型（7 位小数位）
float64 IEEE-754 64位浮点型（15 位小数位）

复数类型：

类型 描述
complex64 32 位实数和虚数
complex128 64 位实数和虚数

其他数字类型：

类型 描述
byte 类似 uint8
rune 类似 int32
uintptr 无符号整型，用于存放一个指针

3. 变量

4. 常量

  • 常量的值在编译时就已经确定，所以常量声明的时候需赋值初始化
  • 常量的定义与全局变量一致
  • 常量定义时，等号右侧必须是常量或常量表达式
  • 常量表达式中的函数必须是内置函数
  • 常量只能定义基本类型

5.类型转换

类型转换包括数字类型之间的转换，也包括数字类型和字符串类型之间转换。数字类型转换需要显式声明转换，字符串转换需要借助strconv包的函数。
