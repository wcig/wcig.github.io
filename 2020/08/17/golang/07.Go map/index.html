Go Map

发表于 2020-08-17 | 分类于 Golang

1. 概念

  • map 是无序键值对
  • 可动态添加和删除键值对
  • make 创建，支持 :=
  • make 创建可指定 cap 容量，添加时超出容量会自动扩容
  • len() 获取键值对个数
  • 通过 range 遍历
  • 不支持定义 const map

2. 初始化

需注意 map 的声明和初始化的区别

3. 添加元素

同一个 key 值，后面的值会覆盖前面的值

4. 删除元素

删除 map 中不存在的 key 不会报错，只是会删除失败

5. 使用range遍历

  • map遍历时，值为真实值的拷贝
  • map 通过 range 修改值一般通过 key
  • 注意 value 为指针的 map
