4. 函数

4.1 Create

os.Create 函数接收一个文件名来创建或截断文件文件，返回值包括文件的描述符和错误。

  • 创建文件时以 0666 权限。
  • 如果文件不存在将创建，如果已经存在会把截断它变成空文件。
  • 返回的文件描述符可以进行 I/O 操作，模式为 O_RDWR。
  • 如果错误，错误底层类型是 *PathError

4.2 Open

os.Open 函数用于打开一个文件去读取，其接收一个文件名，返回文件描述符和错误。对应文件描述符具有 O_RDWR 模式，底层错误类型为 *PathError。

4.3 OpenFile

os.OpenFile 是一个更具有一般性的打开文件的函数。其接收文件名 name，选项 flag（如O_RDWR）和 指定权限（如0666），返回文件的描述符和错误。同样底层错误类型是 *PathError。一般使用 Open、Create 代替该函数。

4.4 Mkdir

os.Mkdir 用于创建目录。其指定目录名和指定权限，如果错误则返回 *PathError 类型错误。

4.5 MkdirAll

os.MkdirAll 同样用于创建目录，与 os.Mkdir 区别在于如果该目录的上级目录不存在时会自动创建。

4.6 Remove

os.Remove 删除文件或目录, 如果出现错误那么错误类型是 *PathError.

4.7 RemoveAll

os.RemoveAll 用于删除目录以及下级对象。它会尝试删除所有文件，除非遇到错误并返回。如果 path 目录不存在将返回 nil 而不是错误。

4.8 Chmod

os.Chmod 类似于 linux 的 chmod 命令，用于修改文件的权限。

4.9 Chown

os.Chown 类似于 linux 的 chown 命令。

Chmod修改name指定的文件对象的用户id和组id。如果name指定的文件是一个符号链接，它会修改该链接的目的地文件的用户id和组id。如果出错，会返回*PathError底层类型的错误。

4.10 Rename

os.Rename 用户重命名文件.

如果 newpath 已经存在或不是一个目录则进行重命名. 可能受操作系统限制.

5. 补充

5.1 Linux文件的access time, change time, modify time

在 Linux 中我们新建一个文件后，可以通过 stat 命令查看文件的信息：

3个时间分别对应：

  • access time：文件最后一次被读取时间
  • change time：文件最后一次元数据被修改（例如权限、位置、用户属性、组属性等）
  • modify time：文件最后一次被修改（文件被修改）

什么情况下3个时间会被修改：

  • access time：cat、touch
  • change time：chmod、chown、mv
  • modify time：vi命令对文件做修改
