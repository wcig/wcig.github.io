Go 反射

发表于 2020-09-19 | 分类于 Golang

1. 反射特性

  • 反射可以提高程序的灵活性, 尤其是对于 interface{}
  • 反射可以获取对象值成员变量属性和值, 也可以获取其公有方法
  • 通过反射可以修改对象的属性值, 也可以动态调用公有方法, 但需要注意值和指针类型

2.3 反射获取方法
  • 可以通过反射可以获取方法, 也可以通过反射调用方法
  • 公有方法可以通过反射获取, 私有方法不行
  • 接收者为值类型方法可以通过反射只能获取值类型方法, 接收者类型为引用方法可以获取值类型和引用类型方法
2.4 反射调用方法
  • 反射不能获取私有方法, 所以不能通过反射调用私有方法
  • 方法接收者为值, reflect.ValueOf 传值
  • 方法接收者为指针, reflect.ValueOf 需传指针

3. 反射总结

3.1 反射函数和类型

go 语言中主要通过 reflec 包提供运行时的反射能力。而反射主要通过 2 个函数来实现：reflect.TypeOf、reflect.ValueOf，其中前者能获取数据的类型信息，后者能获取数据的值信息。

其中 reflect.TypeOf 函数接收一个 interface 类型参数返回一个 Type 类型。而这里的 Type 其实是一个接口，其包含了获取当前类型名、当前类型的方法的引用、当前类型的方法数量等方法：

reflect.ValueOf 函数接收一个 interface 类型参数返回一个 Value 类型值。这里 Value 是一个结构体，包含值的类型信息字段、指向真实数据的指针和一个标识字段，其实现的方法包括设值、获取值等。

3.2 三大法则

go 的反射包括三个法则：

  • go 反射可以将 interface 值转换为反射对象
  • go 反射可以从反射对象获取 interface 值
  • 要想修改一个反射对象，其值必须可被设置

1.将 inerface 值转换为反射对象：

通过 reflect.TypeOf、reflect.ValueOf 可以将 interface 值转换为反射对象，如果参数值本身为非 interface 类型，go 将自动将入参转换为 interface 类型再经过函数转换为对应的反射对象。

2.从反射对象获取 interface 值：

通过 reflect.Value 的 Interface() 方法，可以从反射对象获取 interface 值，再经过类型转换就可得到初始值。

3.要想修改一个反射对象，其值必须可被设置：
